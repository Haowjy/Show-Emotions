The dataset used in this project came from Kaggle (https://www.kaggle.com/c/challenges-in-representation-learning-facial-expression-recognition-challenge) 
and originally from:
"Challenges in Representation Learning: A report on three machine learning 
contests." I Goodfellow, D Erhan, PL Carrier, A Courville, 
M Mirza, B
Hamner, W Cukierski, Y Tang, DH Lee, Y Zhou, C Ramaiah, F Feng, R Li,
 X Wang, D Athanasakis, J Shawe-Taylor, M Milakov, J Park, 
R Ionescu,
 M Popescu, C Grozea, J Bergstra, J Xie, L Romaszko, B Xu, Z Chuang, and
Y. Bengio. arXiv 2013. 

See fer2013.bib for a bibtex entry.


Final FER.py uses kaggle_emotions_model.h5 which has significantly higher detection than emotions_model.h5
trial.py uses emotions_model.h5

open-cv webcam capture: https://docs.opencv.org/3.0-beta/doc/py_tutorials/py_gui/py_video_display/py_video_display.html
open-cv face detection: https://docs.opencv.org/3.4.1/d7/d8b/tutorial_py_face_detection.html
